# 📊 Excel Executive Sales Dashboard — Pivot Tables & Charts

## 📌 Project Overview

This project demonstrates how to build an interactive Executive Sales Dashboard in Microsoft Excel using Pivot Tables, Pivot Charts, Calculated Fields, and Slicers.

The dashboard analyzes sales performance from the Superstore dataset and provides business insights similar to real-world analyst reporting used in companies like Amazon, Myntra, and Deloitte.

---

## 🎯 Objectives

* Summarize large datasets using Pivot Tables
* Analyze regional and category-wise sales
* Track monthly sales trends
* Identify top customers
* Detect low-profit sub-categories
* Build interactive dashboard filters using slicers

---

## 🛠️ Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Calculated Fields

---

## 📊 Dashboard Components

### 1. Regional Performance

* Sum of Sales
* Sum of Profit
* Profit Margin %

### 2. Category Analysis

* Sales by Category and Segment

### 3. Monthly Sales Trend

* Monthly Sales Trend
* Count of Orders

### 4. Top 10 Customers

* Highest revenue-generating customers

### 5. Sub-Category Profitability

* Profitability analysis by sub-category
* Loss-making products identification

---

## 📈 Charts Included

* Clustered Bar Chart → Regional Sales vs Profit
* Line Chart → Monthly Sales Trend
* Bar Chart → Sub-Category Profitability

---

## 🎛️ Interactive Features

* Region Slicer
* Segment Slicer
* Year Timeline Filter

All slicers are connected to multiple Pivot Tables for dynamic filtering.
